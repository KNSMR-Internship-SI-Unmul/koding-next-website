# Project Website Koding Next Samarinda

Pengembangan website profile Koding Next Samarinda bertujuan untuk menyediakan informasi mengenai program dan kegiatan pembelajaran yang ada di Koding Next. Website ini akan menampilkan informasi seperti preview modul pembelajaran, project yang telah dibuat oleh siswa, profil teacher, event yang diselenggarakan, serta profil Koding Next Samarinda.

Selain itu, sistem ini juga dilengkapi dengan dashboard admin yang memungkinkan pihak internal untuk mengelola dan memperbarui konten website secara mudah (CRUD).
